Projekt: Photosnap

Meine Lösung von einem anderen Projekt von der Webseite Frontend-Mentor. Es ist eine 4-seitige Webseite für eine fiktive Plattform, auf der Fotografen ihre Bilder mit anderen teilen können. Die Webseite umfasst eine Startseite und jeweils eine Seite für Stories, Features und Pricing.

Beschreibung

In dieser Challenge von Frontend-Mentor sollte man eine Webseite zum Teilen von Fotos nach Designvorlage realisieren. Auf der Startseite werden die einzelnen Bestandteile der Webseite vorgestellt und verlinkt. Es gibt weiter jeweils eine Seite für Stories, Features und Pricing. Zum Anfang bekommt man von Frontend Mentor den Startercode, der die einzelnen Dateien wie homepage.html oder bookingpage.html enthält. Schon vorgegeben sind lediglich das HTML-Grundgerüst und der Text. Weiter gibt es die verschiedenen Bilder für Handy-, Tablet- und Desktopversion sowie die Designvorlage in einer Figma-Datei.

Die folgenden Anforderungen sollten am Ende erfüllt sein:

  • Optimales Layout für die jeweilige Bildschirmgröße
  • Hoverstates für alle interaktiven Elemente
  • Nutzung von CSS Grid

Tech Stack

HTML5, SCSS, JavaScript

Schwierigkeiten und Lösungen

  1. Dunkelheit von Story-Bildern anpassen: Zuerst habe ich es mit Opacity versucht, aber dann wurde nicht nur das Hintergrundbild, sondern auch die Textfarbe dunkler. Dann mit dem ::before-Pseudoelement, das eine dunklere Schicht auf das Hintergrundbild legt und es somit dunkler macht. Die Originalbilder in der Figma-Designvorlage sind schärfer und anders geschnitten, weshalb ich keine 1-zu-1 Lösung erzielen konnte.
  2. SCSS: Es war das erste Mal, dass ich mit CSS-Präprozessor SCSS gearbeitet habe. Ich fand es schwierig, Verwendung für Features wie Mixins, Operatoren oder Module zu finden. Aus diesem Grund habe ich nur Gebrauch von Variablen und Nesting gemacht, über die ich in der SCSS-Dokumentation gelesen habe. In der Zukunft würde ich allerdings nochmal gerne mit SCSS oder Sass arbeiten und mehr darüber lernen.
  3. CSS-Grids erstellen: Zum Lösen dieser Challenge wurde man dazu ermutigt CSS-Grid zu benutzen. Mit den "normalen" grid-templates hatte ich so meine Schwierigkeiten, weshalb ich mehrmals lieber zu den neueren grid-areas gegriffen habe. Ich finde sie einfacher zu benutzen, weil man einfach in der Vorlage angeben kann, wo jedes Element platziert werden soll. Die andere Variante ist etwas kniffliger, weil genaue Abmessungen für jedes Element angegeben werden müssen.
