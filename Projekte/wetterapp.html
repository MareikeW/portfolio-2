Projekt: Wetter-App

Mit dieser Webapplikation kann nach dem aktuellen Wetter von mehr als 200.000 Orten gesucht werden. Für die Umsetzung des Projekts habe ich die OpenWeatherMap API und die React-Library benutzt.

Beschreibung

Mit dieser Webapplikation kann nach dem aktuellen Wetter von mehr als 200.000 Orten gesucht werden. Sobald die Webseite geladen wurde und sie Zugriff auf den aktuellen Standort des Nutzers hat, wird dessen Ort automatisch als Ausgangspunkt gewählt.

Über ein Eingabefeld lässt sich nach einer anderen Stadt suchen. Bei einer positiven Suche werden die folgenden Angaben unter dem Suchfeld dargestellt:

  • Ort, Land
  • Wochentag, Datum
  • aktuelle Temperatur, gefühlte Temperatur
  • Wetterbeschreibung, passendes Icon

Tech Stack

HTML5, CSS3, JavaScript, React, OpenWeatherMap API

OpenWeatherMap API:
  • Wettervorhersagen in über 40 Sprachen
  • Wetterdaten von über 200.000 Orten auf der Welt
  • Informationen zu Temperatur, Windstärke, Sonnenaufgang und -untergang, etc.

Schwierigkeiten und Lösungen

  1. React ins Projekt integrieren: Dokumentation von ReactJS und VisualStudioCode
  2. Zu großer Codeblock (Anzeige durch CodeMetrics Extension in VSC): Unterteilung in verschiedene Komponenten und einen Context angelegt, um die Zustände query und wetter an mehrere Komponenten weiterzugeben. Normalerweise wird die Context API laut der ReactJS-Dokumentation für das Teilen von beispielsweise Themes oder den derzeitigen authentifizierten Nutzer an verschiedene Komponenten im Hierarchie-Baum. Auslöser für meine Wahl hier war, dass ich erst kurz vor Start des Projekts einen Kurs zu React absolviert hatte. Der Abschnitt zur Context API hat mir dabei am meisten gefallen und ich wollte sie deshalb gleich einmal in einem eigenen Projekt ausprobieren.
  3. Aktuelle Position des Nutzers finden: geolocation-API
  4. Daten von der OpenWeatherMap API abrufen: fetch-API
  5. Wettercontainer ausblenden, wenn keine Wetterdaten vorhanden sind: Dieser Fall tritt immer ein, wenn der Nutzer die Zugriffserlaubnis auf seinen aktuellen Standort (noch) nicht akzeptiert hat.
